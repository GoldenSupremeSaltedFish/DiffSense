# DiffSense Website Skill

## Purpose
This skill provides comprehensive capabilities for creating, managing, and optimizing the official DiffSense website. It focuses on presenting DiffSense as a professional, enterprise-grade code risk analysis tool with clear technical documentation and compelling use cases.

## Key Components

### Technical Content Areas
- **Product Overview**: Clear explanation of DiffSense's core value proposition
- **Technical Architecture**: Detailed breakdown of AST-based semantic analysis, signal-driven architecture, and incremental scanning
- **Use Cases**: Real-world scenarios demonstrating DiffSense's effectiveness in CI/CD pipelines
- **Integration Guides**: Step-by-step instructions for integrating with popular development workflows
- **Performance Benchmarks**: Quantitative data showing efficiency gains over traditional static analysis tools

### Professional Tone Guidelines
- Use precise technical terminology appropriate for developer and DevSecOps audiences
- Avoid overly casual or marketing-heavy language
- Emphasize concrete technical capabilities over vague promises
- Present evidence-based claims with specific examples
- Maintain consistency in technical descriptions across all content

### Visual Design Principles
- Clean, professional layout with clear information hierarchy
- Consistent color scheme reflecting technical sophistication
- Responsive design optimized for both desktop and mobile viewing
- Strategic use of diagrams to illustrate complex architectural concepts
- Code snippets formatted with proper syntax highlighting

## Implementation Files
- `index.html` - Main landing page with responsive design
- `style.css` - Professional styling with consistent typography and spacing
- `script.js` - Interactive elements for enhanced user experience
- `architecture-diagram.svg` - Visual representation of DiffSense's signal-driven architecture
- `integration-examples/` - Directory containing sample integration code and configurations

## Usage
This skill should be used whenever creating or updating content for the DiffSense official website. Always ensure that technical claims are accurate and aligned with the actual capabilities documented in MEMORY.md.
